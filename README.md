Modified Version of the Classic Offline Dino Game

Introduction:

This is a modified version inspired by the original dino game, that you can play on google when you have no wifi.

About:

This game is just like the original game with a slight twist, the dinosaur is changed into a bear inspired by a Chinese cartoon show "Bonnie Bears" and there is also 3 entity excluding the bear, 2 for cactus, and 1 for bird. The unique twist on this game is there is 3 levels, and there is a different music for every level, obstacles are spawned randomly so it's not set, and birds spawn on different score depending on which level you chose.

How to Play:

When you first open this game, you will be prompted with 4 options, but really only the first 3 matters, click accordinly to what is shown on screen to enter the level, and press space bar to jump, when you die, press R to reset, or backspce to exit to the main screen.

Installation:

Copy the code into any IDE for python and it will work
